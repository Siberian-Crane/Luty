# Luty
- Lumitotype专用的命令行压缩工具
    - Lumitotype's dedicated command-line compression tool
- 将文件夹压缩为.ltp格式
    - Compress the folder into an .ltp format file.
- 本质上是压缩为zip格式,然后改后缀名为ltp。您也可以手动改压缩为zip格式并改后缀名
    - Essentially, it compresses to ZIP format and then changes the file extension to .ltp. You can also manually compress it into a ZIP file and rename the extension.
## 安装~Install
- PowerShell
```powershell
irm https://raw.githubusercontent.com/Siberian-Crane/Luty/main/install.ps1 | iex
```
- Bash
```bash
curl -fsSL https://raw.githubusercontent.com/Sberian-Crane/Luty/main/install.sh | bash
```
## 使用~Use
```bash
luty -z 
```
